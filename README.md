# 🎲 Sorteador de Números

Projeto desenvolvido com **HTML, CSS e JavaScript** para sortear números aleatórios sem repetição dentro de um intervalo definido pelo usuário.

## 🚀 Acesse o projeto
👉 https://gabys102.github.io/Sorteador-de-Numero-JS/

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript

## 🧠 Funcionalidades
- Sorteio de números sem repetição
- Definição de quantidade e intervalo
- Validação dos dados informados
- Reinício da aplicação
